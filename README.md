# AI Job Match & Resume Intelligence Platform

An AI-powered platform that compares a candidate's resume with job descriptions.

## Features

- Resume-to-job skill matching
- Match score and missing-skill analysis
- Classical ML job-match prediction
- GenAI resume suggestions
- Interview question generation
- FastAPI backend
- Streamlit dashboard

## Tech Stack

Python, pandas, NumPy, scikit-learn, FastAPI, Streamlit, Gemini API, GitHub
